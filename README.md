# StudentManager
Программа ведет списки студентов.
В ней отображаются id и имя студента.

В ней можно:
- Добавлять новых студентов
- Студенты отображаются в таблице
- Можно удалить выбранного студента из таблицы
  
Чтобы запустить программу, нужно:
- Скачать отсюда (с Github) проект
- Запустить его в Visual Studio
- Нажать на запуск (Run)
(Для работы необходимо наличие базы данных студентов)
